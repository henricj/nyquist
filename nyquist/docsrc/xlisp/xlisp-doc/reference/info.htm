info

Type:   -   function (subr)
Source:   -   xldmem.c

Syntax

(info)
returns - NIL

Description

The 'info' function shows information about memory usage.

Examples

> (info)
[ Free: 5689, GC calls: 17, Total: 675111; samples 1KB, 0KB free]
NIL
