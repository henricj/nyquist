soundp

Type:   -   function (subr)
Source:   -   sndfnint.c, sound.c

Syntax

(soundp expr)
expr - an arbitrary Lisp expression
returns -  T  if expr is a sound, NIL otherwise

Description

The 'soundp' function returns  T  if 'expr' is a Nyquist sound, NIL otherwise.

Examples
