1+

Type:   -   function (subr)
Source:   -   xlmath.c

Syntax

(1+ expr)
expr - integer or floating point number/expression
returns - the number plus one

Description

The '1+' [increment] function adds one to a number and returns the result.

Examples

(1+ 1)     => 2
(1+ 99.1)  => 100.1
(1+ 1 2)   => error: too many arguments
